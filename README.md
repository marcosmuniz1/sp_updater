# sp_viewer
Streamlit app to review searchpattern files
https://spupdater.streamlit.app/

Searchpatterns are a "pointers" within an OTA's booking system that fires queries into aggregator / ndc providers

Due to the manuality involved in searchpattern creation, the app below allows to quickly check how products (travel packages) are being served by the different triggers in the search system. For example, it can be used to see how we are populating flight offers for a London to Egypt travel offer

Sample files are provided for illustrative purposes
